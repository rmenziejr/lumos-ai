# Lumosai v0.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add v0.2 support for representative samples, metadata-first dataset logging, feature importance, and pipeline-oriented docs without turning `lumosai` into an orchestrator.

**Architecture:** Keep the function-first API and the existing `LumosResult` return contract. Add a focused `lumosai.data.sampling` module for bounded representative samples, a focused `lumosai.model.importance` module for feature importance, and small MLflow helpers for dataset metadata/artifact logging. Treat pipeline structure as docs and recipes, not runtime window management.

**Tech Stack:** Python 3.11, pandas, narwhals ingestion through `to_pandas`, scikit-learn permutation importance, optional SHAP through `lumosai[importance]`, Pydantic settings, MLflow optional dependency, pytest, ruff, mypy, MkDocs.

## API Decisions

`build_sample()` is a data helper, not a scheduler:

```python
def build_sample(
    data: Any,
    *,
    role: Literal["train_benchmark", "holdout", "monitoring_window"],
    sample_size: int | None = None,
    strategy: Literal["auto", "random", "stratified", "temporal_recent", "temporal_bucket"] = "auto",
    target: str | None = None,
    prediction: str | None = None,
    feature_columns: list[str] | None = None,
    categorical_columns: list[str] | None = None,
    time_column: str | None = None,
    temporal_columns: list[str] | None = None,
    stratify_by: str | list[str] | None = None,
    random_state: int = 42,
    artifact_path: str | Path | None = None,
    experiment_name: str | None = None,
    log_metadata: bool = True,
    log_artifact: bool = False,
) -> LumosResult:
```

Role defaults:
- `train_benchmark`: complete feature representation for drift baseline; excludes `time_column` and `temporal_columns` from sampled output by default; uses stratified sampling when `target` or `stratify_by` is available, otherwise random.
- `holdout`: model performance sample; if `time_column` exists and no strategy is provided, uses most recent rows; includes `target`, `prediction`, and selected features when present.
- `monitoring_window`: bounded production sample; if `time_column` exists and no strategy is provided, uses most recent rows; if `sample_size` is omitted, returns the bounded frame after column selection.

`feature_importance()` is a model helper:

```python
def feature_importance(
    model: Any,
    data: Any,
    *,
    target: str,
    feature_columns: list[str],
    method: Literal["permutation", "shap"] = "permutation",
    scoring: str | Callable[..., float] | None = None,
    n_repeats: int = 5,
    sample_size: int | None = None,
    random_state: int = 42,
    report_name: str | None = None,
    experiment_name: str | None = None,
) -> LumosResult:
```

MLflow recommendation:
- Default: log dataset metadata only with `mlflow.log_dict`, including role, row count, column list, schema, digest, time range, strategy, and local or external URI.
- Opt-in: log raw sample file as an artifact only when `log_artifact=True`, because MLflow artifact stores may not be approved for representative datasets.
- Defer MLflow `Dataset` API integration until the exact MLflow version and desired dataset UI behavior are validated.

Create `src/lumosai/data/sampling.py`:

```python
from __future__ import annotations

import hashlib
from pathlib import Path
from typing import Any, Literal

import pandas as pd

from lumosai.data.ingest import to_pandas
from lumosai.data.validation import require_columns
from lumosai.exceptions import LumosValidationError
from lumosai.mlflow import log_sample
from lumosai.results import LumosResult
from lumosai.schema import select_analysis_frame, validate_categorical_columns
from lumosai.settings import settings

SampleRole = Literal["train_benchmark", "holdout", "monitoring_window"]
SampleStrategy = Literal["auto", "random", "stratified", "temporal_recent", "temporal_bucket"]


def _digest(frame: pd.DataFrame) -> str:
    hashed = pd.util.hash_pandas_object(frame, index=False).values.tobytes()
    return hashlib.sha256(hashed).hexdigest()


def _schema(frame: pd.DataFrame) -> dict[str, str]:
    return {column: str(dtype) for column, dtype in frame.dtypes.items()}


def _time_range(frame: pd.DataFrame, time_column: str | None) -> dict[str, str] | None:
    if time_column is None or time_column not in frame.columns:
        return None
    values = pd.to_datetime(frame[time_column], errors="raise")
    return {"min": values.min().isoformat(), "max": values.max().isoformat()}


def _selected_columns(
    frame: pd.DataFrame,
    *,
    role: SampleRole,
    target: str | None,
    prediction: str | None,
    feature_columns: list[str] | None,
    time_column: str | None,
    temporal_columns: list[str],
) -> list[str]:
    required = [column for column in [time_column, prediction] if column is not None]
    selected = select_analysis_frame(
        frame,
        target=target,
        feature_columns=feature_columns,
        required_columns=required,
    )
    columns = list(selected.columns)
    if role == "train_benchmark":
        temporal_exclusions = {column for column in [time_column, *temporal_columns] if column}
        columns = [column for column in columns if column not in temporal_exclusions]
    return columns


def _resolve_strategy(
    *,
    role: SampleRole,
    strategy: SampleStrategy,
    time_column: str | None,
    target: str | None,
    stratify_by: str | list[str] | None,
) -> SampleStrategy:
    if strategy != "auto":
        return strategy
    if role in {"holdout", "monitoring_window"} and time_column is not None:
        return "temporal_recent"
    if role == "train_benchmark" and (target is not None or stratify_by is not None):
        return "stratified"
    return "random"


def _sample_random(frame: pd.DataFrame, sample_size: int | None, random_state: int) -> pd.DataFrame:
    if sample_size is None or sample_size >= len(frame):
        return frame.copy()
    return frame.sample(n=sample_size, random_state=random_state).sort_index().reset_index(drop=True)


def _sample_recent(frame: pd.DataFrame, time_column: str, sample_size: int | None) -> pd.DataFrame:
    ordered = frame.sort_values(time_column)
    if sample_size is None or sample_size >= len(ordered):
        return ordered.reset_index(drop=True)
    return ordered.tail(sample_size).reset_index(drop=True)


def _sample_stratified(
    frame: pd.DataFrame,
    *,
    sample_size: int | None,
    stratify_columns: list[str],
    random_state: int,
) -> pd.DataFrame:
    if sample_size is None or sample_size >= len(frame):
        return frame.copy().reset_index(drop=True)
    require_columns(frame, stratify_columns)
    fractions = sample_size / len(frame)
    sampled = (
        frame.groupby(stratify_columns, group_keys=False, dropna=False)
        .sample(frac=fractions, random_state=random_state)
        .reset_index(drop=True)
    )
    if len(sampled) < sample_size:
        remaining = frame.drop(sampled.index, errors="ignore")
        top_up = remaining.sample(n=sample_size - len(sampled), random_state=random_state)
        sampled = pd.concat([sampled, top_up], ignore_index=True)
    return sampled.head(sample_size).reset_index(drop=True)


def _apply_strategy(
    frame: pd.DataFrame,
    *,
    strategy: SampleStrategy,
    sample_size: int | None,
    time_column: str | None,
    target: str | None,
    stratify_by: str | list[str] | None,
    random_state: int,
) -> pd.DataFrame:
    if strategy == "temporal_recent":
        if time_column is None:
            msg = "time_column is required for temporal_recent sampling"
            raise LumosValidationError(msg)
        return _sample_recent(frame, time_column, sample_size)
    if strategy == "temporal_bucket":
        if time_column is None:
            msg = "time_column is required for temporal_bucket sampling"
            raise LumosValidationError(msg)
        from lumosai.data.profiling import temporal_sample

        rows_per_period = settings.data.default_sample_size if sample_size is None else sample_size
        return temporal_sample(frame, time_column=time_column, sample_size=rows_per_period)
    if strategy == "stratified":
        columns = stratify_by if stratify_by is not None else target
        stratify_columns = [columns] if isinstance(columns, str) else list(columns or [])
        if not stratify_columns:
            msg = "stratified sampling requires target or stratify_by"
            raise LumosValidationError(msg)
        return _sample_stratified(
            frame,
            sample_size=sample_size,
            stratify_columns=stratify_columns,
            random_state=random_state,
        )
    return _sample_random(frame, sample_size, random_state)


def build_sample(
    data: Any,
    *,
    role: SampleRole,
    sample_size: int | None = None,
    strategy: SampleStrategy = "auto",
    target: str | None = None,
    prediction: str | None = None,
    feature_columns: list[str] | None = None,
    categorical_columns: list[str] | None = None,
    time_column: str | None = None,
    temporal_columns: list[str] | None = None,
    stratify_by: str | list[str] | None = None,
    random_state: int = 42,
    artifact_path: str | Path | None = None,
    experiment_name: str | None = None,
    log_metadata: bool = True,
    log_artifact: bool = False,
) -> LumosResult:
    if sample_size is not None and sample_size < 1:
        msg = "sample_size must be greater than zero"
        raise LumosValidationError(msg)
    if role == "holdout" and prediction is not None and target is None:
        msg = "target is required when prediction is provided for holdout samples"
        raise LumosValidationError(msg)

    frame = to_pandas(data)
    temporal = list(temporal_columns or [])
    required = [column for column in [target, prediction, time_column, *temporal] if column is not None]
    if feature_columns is not None:
        required.extend(feature_columns)
    require_columns(frame, required)

    columns = _selected_columns(
        frame,
        role=role,
        target=target,
        prediction=prediction,
        feature_columns=feature_columns,
        time_column=time_column,
        temporal_columns=temporal,
    )
    selected_categorical = validate_categorical_columns(
        frame,
        categorical_columns=categorical_columns,
        analysis_columns=columns,
    )
    resolved_strategy = _resolve_strategy(
        role=role,
        strategy=strategy,
        time_column=time_column,
        target=target,
        stratify_by=stratify_by,
    )
    sampled = _apply_strategy(
        frame,
        strategy=resolved_strategy,
        sample_size=sample_size,
        time_column=time_column,
        target=target,
        stratify_by=stratify_by,
        random_state=random_state,
    )[columns]

    summary: dict[str, Any] = {
        "role": role,
        "strategy": resolved_strategy,
        "source_rows": len(frame),
        "sample_rows": len(sampled),
        "columns": list(sampled.columns),
        "categorical_columns": selected_categorical,
        "schema": _schema(sampled),
        "digest": _digest(sampled),
    }
    time_range = _time_range(sampled, time_column)
    if time_range is not None:
        summary["time_range"] = time_range

    result = LumosResult(
        summary=summary,
        artifacts={"sample": sampled},
        metadata={"report_type": "sample", "sample_role": role},
    )
    log_sample(
        result,
        artifact_path=artifact_path,
        experiment_name=experiment_name,
        log_metadata=log_metadata,
        log_artifact=log_artifact,
    )
    return result
```

```python
def _write_sample_artifact(frame: Any, path: Path) -> Path:
    path.parent.mkdir(parents=True, exist_ok=True)
    if path.suffix == ".csv":
        frame.to_csv(path, index=False)
        return path
    frame.to_parquet(path, index=False)
    return path


def log_sample(
    result: LumosResult,
    *,
    artifact_path: str | Path | None = None,
    experiment_name: str | None = None,
    loaded_settings: Settings = settings,
    log_metadata: bool | None = None,
    log_artifact: bool | None = None,
) -> LumosResult:
    sample = result.artifacts.get("sample")
    should_log_metadata = loaded_settings.data.log_sample_metadata if log_metadata is None else log_metadata
    should_log_artifact = loaded_settings.data.log_sample_artifacts if log_artifact is None else log_artifact

    if sample is not None and artifact_path is not None:
        path = _write_sample_artifact(sample, Path(artifact_path))
        result.metadata["sample_artifact_path"] = str(path)

    resolved = resolve_experiment_name(experiment_name, loaded_settings)
    if resolved is None:
        result.metadata["logged_to_mlflow"] = False
        return result

    with mlflow_run(resolved, loaded_settings) as (mlflow, run_id):
        if mlflow is None:
            result.metadata["logged_to_mlflow"] = False
            return result
        result.metadata["logged_to_mlflow"] = True
        result.metadata["mlflow_run_id"] = run_id
        if should_log_metadata:
            mlflow.log_dict(result.summary, "lumosai_sample_metadata.json")
        if should_log_artifact and artifact_path is not None:
            mlflow.log_artifact(str(artifact_path), artifact_path="samples")
    return result
```

Create `src/lumosai/model/importance.py`:

```python
from __future__ import annotations

from collections.abc import Callable
from typing import Any, Literal

import pandas as pd
from sklearn.inspection import permutation_importance as sklearn_permutation_importance

from lumosai.data.ingest import to_pandas
from lumosai.data.validation import require_columns
from lumosai.exceptions import LumosOptionalDependencyError, LumosValidationError
from lumosai.mlflow import log_result
from lumosai.results import LumosResult
from lumosai.settings import settings

ImportanceMethod = Literal["permutation", "shap"]


def _sample_frame(frame: pd.DataFrame, sample_size: int | None, random_state: int) -> pd.DataFrame:
    if sample_size is None or sample_size >= len(frame):
        return frame
    if sample_size < 1:
        msg = "sample_size must be greater than zero"
        raise LumosValidationError(msg)
    return frame.sample(n=sample_size, random_state=random_state).reset_index(drop=True)


def _permutation_importance(
    model: Any,
    frame: pd.DataFrame,
    *,
    target: str,
    feature_columns: list[str],
    scoring: str | Callable[..., float] | None,
    n_repeats: int,
    random_state: int,
) -> tuple[dict[str, float], list[dict[str, float | str]]]:
    result = sklearn_permutation_importance(
        model,
        frame[feature_columns],
        frame[target],
        scoring=scoring,
        n_repeats=n_repeats,
        random_state=random_state,
    )
    rows = [
        {
            "feature": feature,
            "importance_mean": float(mean),
            "importance_std": float(std),
        }
        for feature, mean, std in zip(
            feature_columns,
            result.importances_mean,
            result.importances_std,
            strict=True,
        )
    ]
    rows.sort(key=lambda row: float(row["importance_mean"]), reverse=True)
    metrics = {f"importance/{row['feature']}": float(row["importance_mean"]) for row in rows}
    return metrics, rows


def _shap_importance(
    model: Any,
    frame: pd.DataFrame,
    *,
    feature_columns: list[str],
) -> tuple[dict[str, float], list[dict[str, float | str]]]:
    try:
        import shap
    except ImportError as exc:
        msg = "SHAP feature importance requires the optional dependency: pip install lumosai[importance]"
        raise LumosOptionalDependencyError(msg) from exc

    explainer = shap.Explainer(model, frame[feature_columns])
    values = explainer(frame[feature_columns])
    raw_values = values.values
    if raw_values.ndim == 3:
        raw_values = abs(raw_values).mean(axis=(0, 2))
    else:
        raw_values = abs(raw_values).mean(axis=0)
    rows = [
        {"feature": feature, "importance_mean": float(value), "importance_std": 0.0}
        for feature, value in zip(feature_columns, raw_values, strict=True)
    ]
    rows.sort(key=lambda row: float(row["importance_mean"]), reverse=True)
    metrics = {f"importance/{row['feature']}": float(row["importance_mean"]) for row in rows}
    return metrics, rows


def feature_importance(
    model: Any,
    data: Any,
    *,
    target: str,
    feature_columns: list[str],
    method: ImportanceMethod = "permutation",
    scoring: str | Callable[..., float] | None = None,
    n_repeats: int = 5,
    sample_size: int | None = None,
    random_state: int = 42,
    report_name: str | None = None,
    experiment_name: str | None = None,
) -> LumosResult:
    if n_repeats < 1:
        msg = "n_repeats must be greater than zero"
        raise LumosValidationError(msg)
    frame = _sample_frame(to_pandas(data), sample_size, random_state)
    require_columns(frame, [target, *feature_columns])

    if method == "permutation":
        metrics, rows = _permutation_importance(
            model,
            frame,
            target=target,
            feature_columns=feature_columns,
            scoring=scoring,
            n_repeats=n_repeats,
            random_state=random_state,
        )
    else:
        shap_sample_size = sample_size or settings.model.shap_sample_size
        shap_frame = _sample_frame(frame, shap_sample_size, random_state)
        metrics, rows = _shap_importance(model, shap_frame, feature_columns=feature_columns)
        frame = shap_frame

    metadata: dict[str, Any] = {
        "report_type": "feature_importance",
        "method": method,
        "feature_columns": list(feature_columns),
    }
    if report_name is not None:
        metadata["report_name"] = report_name

    result = LumosResult(
        metrics=metrics,
        summary={"rows": len(frame), "features": rows},
        metadata=metadata,
    )
    log_result(result, experiment_name=experiment_name)
    return result
```

````markdown
# Pipeline Patterns

`lumosai` is a reporting SDK. Your orchestrator should own scheduling, joins, storage, and retries.

## Data pipeline

1. Build or refresh feature tables.
2. Profile processed features.
3. Compare the new feature frame against a train benchmark with `drift_report()`.
4. Log results to the pipeline's MLflow run.

## Training pipeline

1. Train the model.
2. Build a train benchmark sample with `build_sample(role="train_benchmark")`.
3. Build a holdout sample with `build_sample(role="holdout")`.
4. Run `performance_report()`, `bias_report()` when permitted, and `feature_importance()`.
5. Log sample metadata to MLflow. Log raw sample artifacts only when approved for the data.

## Ongoing monitoring pipeline

1. Build a bounded production monitoring window upstream.
2. Run drift against the train benchmark.
3. Optionally run drift against the previous production window.
4. Run performance when labels arrive.
5. Run bias when protected attributes are available and approved for monitoring.

Drift comparisons should use distinct `comparison` labels such as `benchmark` and `previous_window`. Performance and bias do not need comparison labels because trend history belongs to MLflow runs over time.
````
