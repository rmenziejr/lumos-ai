# Example Data

## `diabetes.csv`

This example dataset is the Pima Indians Diabetes Database downloaded from Kaggle:

<https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database>

It is included so the walkthrough notebook can run without Kaggle credentials or a
network download. The notebook adds derived columns such as `encounter_date`,
`age_group`, `prediction`, and probability columns at runtime; the checked-in CSV is
the source dataset from the Kaggle download.
